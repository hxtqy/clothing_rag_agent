# 黑马衣物 Agent：智能穿搭客服系统
基于 **RAG（检索增强生成）+ Streamlit** 开发的衣物穿搭智能客服 Agent，为用户提供专业的尺码推荐、洗涤养护、颜色搭配等咨询服务，支持多会话记忆与知识库动态更新。

## 🌟 核心功能
- **智能问答**：结合知识库，精准回答用户关于衣物尺码、洗涤、颜色搭配等穿搭问题
- **多会话管理**：支持创建、切换、删除会话，持久化对话历史，重启后仍可恢复
- **知识库管理**：上传 TXT 文件即可更新知识库，自动向量化存储，无需手动处理数据
- **流式回答**：模拟打字机效果，提升对话交互体验
- **安全可靠**：会话历史本地存储，API 密钥环境变量管理，避免敏感信息泄露

## 📁 核心文件说明
| 文件名 | 功能描述 |
|--------|----------|
| `app_file_upload.py` | 知识库更新主程序（基于 Streamlit），提供可视化界面上传 TXT 文件并更新向量知识库 |
| `app_qa.py` | 项目主程序，启动对话 WEB 页面，是用户与智能客服交互的入口 |
| `config_data.py` | 全局配置文件，管理模型名称、存储路径、API 配置等全局参数 |
| `file_history_store.py` | 长期会话记忆存储服务，基于文件系统持久化对话历史，支持多会话隔离 |
| `knowledge_base.py` | 知识库更新核心服务，负责文件解析、文本分块、向量化与向量库写入 |
| `rag.py` | RAG 核心服务，整合向量检索、Prompt 模板构建、LLM 调用与流式输出逻辑 |
| `vector_stores.py` | 向量存储服务，封装与 Chroma 向量库的交互，管理向量数据的增删查 |
